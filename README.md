# HireFlow — Диагностика бизнеса

Интерактивный квиз для диагностики проблем в привлечении, конверсии и удержании клиентов.

## Запуск

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Структура проекта

```
hr-quiz/
├── data/
│   └── quiz.json              # Все вопросы, варианты и результаты
├── components/
│   ├── QuizContainer.tsx      # Основная логика квиза
│   ├── Question.tsx           # Компонент вопроса
│   ├── ProgressBar.tsx        # Прогресс-бар
│   └── Results.tsx            # Экран с результатами
├── hooks/
│   └── useQuizState.ts        # Управление состоянием (Local Storage)
├── pages/
│   ├── index.tsx              # Главная страница
│   └── _document.tsx          # HTML шаблон
├── styles/
│   └── globals.css            # Глобальные стили
├── public/                    # Статические файлы
└── next.config.js             # Конфигурация Next.js
```

## Как работает

1. **Выбор типа бизнеса** — пользователь указывает, какой у него бизнес (кофейня, бьюти, услуги, розница или другое)
2. **Выбор этапа** — новый, растущий или установившийся бизнес
3. **Выбор области диагностики** — привлечение, конверсия или удержание клиентов
4. **Ответы на вопросы** — адаптивные вопросы в зависимости от ветки
5. **Результаты** — подробная диагностика с рекомендациями и экшн-планом

## Состояние

Состояние квиза сохраняется в `localStorage`, поэтому пользователь может вернуться и продолжить позже.

## Кастомизация

### Добавление новых вопросов
Отредактируй `/data/quiz.json` и добавь новый вопрос в соответствующую ветку.

### Изменение цветов
Бордо: `#8B3A42`, Тёмный синий: `#1E3A5F`, Тёмный зелёный: `#2D5A3D`

### Добавление новых результатов
Добавь новый тег в `diagnosticResults` в `/data/quiz.json`.

## Деплой

Приложение готово к деплою на [Vercel](https://vercel.com):

```bash
npm run build
```

Затем экспортируй на Vercel через CLI или веб-интерфейс.
